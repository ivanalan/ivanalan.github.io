River Drifter

Media Device Architectures Project

January 2020 - June 2020

In Fall 2019 I took a class called Media Device Architectures where I learned fundamentals of C. I learned the following concepts in C:

  • Using page flipping / double buffering for smooth animation
  • Creating simple animations used in simple bit-mapped games
  • Using Dynamic Memory Access to speed up animations
  • Maintaining and changing program state

For my final project, I created an original GameBoy Advance game in C and implemented unique features such as parallax movement, animated sprites, and a cheat code. Look at my code here
